So remove the older versions of your kernel after each upgrade. Else, you wont be able to start a virtual OS by hitting start. You will get an error saying run "/etc/init.d/vboxdrv setup".
To see the list of kernels available, type rpm -qa kernel. To remove older kernels, type rpm -e kernel-<version> Then run /etc/init.d/vboxdrv setup
Now your virtual os will start normally.
